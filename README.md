# Pizza Tower FMOD Studio Project Recreation [Mixer Accuracy Rev.]

[![PT FSPRO Repository Image](.github/readme/banner-fspro.png)](https://gamebanana.com/tools/13432)

A rather unorthodox fork of the Pizza Tower FSPRO Recreation Project that aims to match the mixer bus to the original's,  allowing you to use modded music and sfx banks with the original Master banks.

If you're lost or looking for instructions on how to use this,  refer to the original GitHub that this is forked from!
